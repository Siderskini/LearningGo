# General tools for go games
Through development of games, this collection of files contains generic building blocks for games, including:
input:
    - handling for direction of a swipe input
    - handling for clicks and keypresses
    - handling for textinput
    - buttons (circle, rectangle)
    - scrolling menus (vertical)

a/v:
    - playing of gifs
    - playing of wav files
    - handling of icons on buttons
    - fonts
    - colors (needs some work)

functional:
    - saving (file, browser storage)
